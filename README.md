# 📊 Telecom User Analytics Project

## 🔹 Features
- User Overview Analysis (EDA)
- Engagement Analysis (Clustering)
- Experience Analytics (QoS Metrics)
- Satisfaction Modeling
- Model Tracking (MLOps)
- Streamlit Dashboard

streamlit run app.py

- model_tracking.csv

## 🎯 Insights
- High engagement users generate most traffic
- QoS strongly impacts satisfaction
- Video apps dominate network usage
